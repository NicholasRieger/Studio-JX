# Studio JX

Landing page desenvolvida para um estúdio de estética, com foco em apresentação visual, organização dos serviços, clareza das informações e contato facilitado pelo WhatsApp.

O projeto foi pensado para transmitir uma imagem mais elegante e profissional da marca, valorizando a experiência no celular e a navegação simples.

## Visão geral

O site foi construído como uma página de apresentação comercial para divulgar os serviços do estúdio e facilitar o agendamento.

Entre os principais pontos do projeto estão:

- hero com proposta visual mais delicada e comercial
- seções de procedimentos, pacotes, sobre e contato
- navegação responsiva
- foco em leitura, hierarquia visual e experiência mobile
- botão de agendamento direto pelo WhatsApp

## Tecnologias utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome

## Objetivo do projeto

Este projeto foi desenvolvido para:

- apresentar os serviços do estúdio de forma mais profissional
- melhorar a comunicação visual da marca
- organizar as informações de forma mais clara
- facilitar o contato e o agendamento
- Destaques do projeto
- layout responsivo
- visual suave e coerente com a proposta do estúdio
- estrutura pensada para apresentação comercial
- CTA direto para WhatsApp
- cuidado com espaçamento, tipografia e hierarquia visual
